
# Password Manager

A beginner-friendly console-based Password Manager built in C#.

---

## About This Project

This is a simple Password Manager application built in C#
for beginners. The user is protected by a master password
at login. All saved passwords are encrypted so they are
not stored as plain text. It runs fully in the console
with no extra tools or database required.

This project was built to practice C# concepts like
functions, lists, loops, and basic encryption.

---

## Language and Technology

- Language  : C# (C-Sharp)
- Platform  : .NET Console Application
- File      : Single file (Program.cs)

---

## Concepts and Techniques Used

- Object Oriented Programming (OOP)
- Functions and Methods
- Lists (Collections)
- Loops (while, for, foreach)
- Conditional Statements (if-else, switch)
- XOR Encryption for password security
- Base64 Encoding
- String Manipulation
- Type Conversion (int.TryParse)
- Random Number Generation

---

## Features

- Master password protection at login
- Add password with website and username
- View all saved passwords
- Search password by website name
- Delete saved password
- Generate strong random password
- Basic encryption so passwords are not stored as plain text

---

## How to Run

### Online (No installation needed)
1. Go to dotnetfiddle.net
2. Create a new C# console project
3. Paste the code and click Run

### On Laptop or Desktop
1. Install .NET SDK from dotnet.microsoft.com
2. Open terminal and create new project
   dotnet new console -n PasswordManager
3. Paste the code in Program.cs
4. Run using command
   dotnet run
